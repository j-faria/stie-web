About this site¶

This small webpage serves as a companion for my Master Thesis entitled “Asteroseismology of 16 Cyg A and B”.

It exists because of my desire to make publicly available the code and data that I used to produce the results described in my thesis. My reasons to do so are well stated in the Science Code Manifesto. “Code is method”.

This webpage is referenced in the final version of the thesis, presented to the University of Porto, at DATE. You can get the pdf here (size).

Software and Input files¶

MESA models¶

According to the MESA website: “Before you use MESA, you should at least skim the ‘instrument papers’.” The code is open source and available for download. Instructions on how to do it can be found here.

The way to talk to MESA is by preparing the so called ‘inlist’ files, which are parameter files that are read by the code at runtime, setting up all the necessary options to do a stellar evolution run.

We have used two inlist files to perform the stellar modelling of Section 4.3, one for each of 16 Cyg A and B
